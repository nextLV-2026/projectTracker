프로젝트 GitHub 레포지토리 연결 *
워크플로우 자동 업데이트의 핵심 — 커밋과 PR을 지속적으로 받아와 진척도를 반영합니다
레포지토리 URL을 입력하면 연결 상태가 표시됩니다
⚡ 자동 동기화 항목
  • 커밋 메시지 분석 → 완료된 태스크 자동 체크
  • PR 생성 / 머지 → 해당 태스크 상태 자동 변경 (In Progress → Review → Done)
  • 브랜치 현황 → 팀원별 작업 진척도 실시간 반영
  • 이슈 트래킹 → 새 이슈 생성 시 워크플로우에 태스크 자동 추가
💡 연결 후 Project Tracker가 주기적으로 레포지토리를 폴링합니다. 실시간 반영을 원하면 GitHub Webhook 설정을 권장합니다. (AI 분석 완료 후 안내됩니다)
